Git Command & Description

1. git init

initialize the git to the project directory

2. git status

To check the untracked and tracked file status before commiting or adding to git

3. git add filename

to add single file to git cache if changes is done

4. git add .

to add all files to git cache if changes is done

5. git rm --cached filename

to remove the added file from git cache

6. git restore filename

to remove all new changes if the changes are not added to git cache

7. git commit -m "Message"

To add to git file. here -m is use to add message for commit

8. git remote add origin https/ssh repo link

to add the remote server repository to local server

9. git show

to check the file content

10. git push -u origin branchname

to push our changes to remote server

If we want to push in master then branchname = master
